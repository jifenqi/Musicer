# musicer

## 安装依赖
```
npm install
```

### 开发环境启动服务
```
npm run serve
```

### 打包 默认为生产环境（压缩）
```
npm run build
```

### 后台接口服务
```
网易云API 接口文档 (https://binaryify.github.io/NeteaseCloudMusicApi)
```

### 项目中的插件
```
    axios          #发送ajax请求
    better-scroll  #滚动组件
    vue-lazyload   #图片懒加载组件
```
